# Logg
这是一个Android 轻量级简单易用的Log组件

## Features  
* 支持输出基本数据类型、数组、Map、List、Intent、Bundle，支持Json、XML格式化输出  
* 自定义解析器，当输入类型无法满足需求时或无法解析时，可实现Parser接口重写方法来添加自定义解析器
